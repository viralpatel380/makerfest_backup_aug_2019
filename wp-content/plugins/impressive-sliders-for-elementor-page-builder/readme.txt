Image slider and gallery elements for Elementor page builder using jQuery Jssor Slider.

== Description ==

[Impressive Sliders for Elementor Page Builder](http://www.wpexpertdeveloper.com/impressive-sliders-elementor-page-builder/) is an awesome image slider and gallery elements for Elementor Page Builder. Sliders are powered by popular open source Jssor jQuery Slider. Provides wide range of transition effects to manage your image sliders.

This is an addon for Elementor Page Builder. You need to install Elementor version 1.8.11 or higher before using this plugin.


> **Features**
>
> * Simple and Flexible Image Slider elements
> * Image Gallery elements
> * 10 Awesome Slide Transition Effects

== Installation ==

1. Upload the plugin files to the `/wp-content/plugins/impressive-sliders-elementor-page-builder` directory, or install the plugin through the WordPress plugins screen directly.
2. Activate the plugin through the 'Plugins' screen in WordPress
3. Use Impressive Slider element in Elementor Page Builder to create sliders.
4. Follow the <a href="http://www.wpexpertdeveloper.com/impressive-sliders-elementor-page-builder/">Documentation and Usage</a>.
